# Web App Cloud Template
Run a (static) web application in a cloud service.

In this project, the following are used:

| Tool                   | Name                                         |
|------------------------|----------------------------------------------|
| Cloud Service          | **AWS** (Only S3 and CloudFront for the demo)|
| CI/CD                  | **CircleCI**                                 |
| Web                    | **index.html**                               |
| Infrastructure as Code | **Terraform**                                |
